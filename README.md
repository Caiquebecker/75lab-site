# 75 LAB — Site institucional

Aceleradora de trade marketing. Site estático (HTML/CSS/JS), multi-páginas, sem build.

**Slogan:** _ideia boa é a que acontece_

## Páginas
| Arquivo | Página |
|---|---|
| `index.html` | Home |
| `solucoes.html` | Soluções (hub + método) |
| `cases.html` | Cases |
| `sobre.html` | Sobre (origem + fundadores) |
| `midia.html` | Mídia (imprensa + redes) |
| `contato.html` | Contato |
| `404.html` | Página de erro |

## Arquivos compartilhados
- `style.css` — todo o visual (edite o bloco `:root` para cores/fontes)
- `app.js` — todas as animações e efeitos

## Como rodar localmente
Abra o `index.html` no navegador. Não precisa de servidor nem build.

## Publicar
- **Hostinger:** suba todos os arquivos para `public_html`.
- **GitHub Pages:** Settings → Pages → Branch `main` /root.

Veja `LEIA-ME.txt` para o passo a passo de edição (cases, logos, imprensa, redes, fotos).
